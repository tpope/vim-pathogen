This project adds [pathogen] to the vim editor. It manages your runtime path.

[pathogen]: http://www.vim.org/scripts/script.php?script_id=2332

### Installing

```bash
curl https://raw.github.com/anirudhjoshi/vim-pathogen/master/install.sh | sh
```

That's it.

### Updating

```bash
curl https://raw.github.com/anirudhjoshi/vim-pathogen/master/update.sh | sh
```

Everything will then be brought up to date.

### Uninstalling

```bash
curl https://raw.github.com/anirudhjoshi/vim-pathogen/master/uninstall.sh | sh
```

vim-pathogen will be uninstalled.
